## **1. 게임 컨셉 (High Concept)**
* **High Concept**
    * **대시(Dash)** 와 **밟기(Stomp)** 를 통해 적을 밀어내며 전진하는 액션 게임입니다.
* **핵심 메카닉**:
    * 화면 터치 시 전방으로 돌진하여 적을 강력하게 밀쳐냄
    * 낙하 중 적의 머리를 밟으면 점프
    * 미리 설계된 발판 패턴들을 무작위로 조합해 **Object Recycle**을 통해 무한한 지형을 생성

## **2. 개발 범위 (정량적 제시)**

### **[캐릭터 및 애니메이션]**
* 플레이어 캐릭터 **1종** 구현
* 대기, 대시, 점프, 충돌 시 반동 등 상태별 **Animation** 적용

### **[물리 및 충돌 시스템]**
* **Velocity Control**: **Choreographer** 기반의 프레임 제어를 통해 대시 가속도 및 중력 물리 연산 적용
* **Collision Handling**: 충돌 시점의 상대 좌표와 속도 벡터를 계산하여 **'옆으로 밀기'** 와 **'위에서 밟기'** 판정 분기

### **[적 및 지형 환경]**
* 몬스터 **2종**(일반형, 거대형) 구현
* **HorzScroll BG**를 활용한 다중 레이어 배경 연출
* **Object Lifecycle (recycle)** 기반의 무한 발판 생성 및 메모리 최적화


## **3.  예상 게임 실행 흐름**
![Image](https://github.com/user-attachments/assets/497e52c6-193c-4140-8b65-bce09f4c6e2f)
* **메인 화면**: 게임 제목과 시작 버튼이 위치
* **인게임**: 캐릭터는 자동으로 짧은 점프를 반복하며 대기. 터치 시 전방으로 대시하며 스크롤이 가속됩니다.
* **전투 액션**: 
    * 대시로 적을 들이받아 발판 밖으로 밀어내어 제거.
    * 적의 머리 위로 점프해 올라가 장애물을 통과.
* **결과 화면**: 캐릭터가 발판 사이로 추락하거나 적에게 눌릴 경우 최종 점수를 표시 후 재시작
